1. Als nieuwe gebruiker wil ik een account kunnen aanmaken, zodat ik mijn voortgang en saldo kan bewaren.
Prioriteit: Must
Acceptatiecriteria:
- Gebruiker kan registreren met e‑mail + wachtwoord.
- Validatie toont duidelijke foutmeldingen bij ongeldige invoer.
- Account wordt opgeslagen en is direct bruikbaar.
- Gebruiker wordt automatisch ingelogd na registratie.

2. Als gebruiker wil ik kunnen inloggen en uitloggen, zodat ik veilig toegang heb tot mijn account.
Prioriteit: Must
Acceptatiecriteria:
- Inloggen werkt met correcte e‑mail + wachtwoord.
- Foutieve gegevens geven een duidelijke melding.
- Uitloggen verwijdert sessiegegevens.
- Sessies verlopen automatisch na een bepaalde tijd.

3. Als gebruiker wil ik mijn virtuele saldo kunnen bekijken, zodat ik weet hoeveel ik kan spelen.
Prioriteit: Must
Acceptatiecriteria:
- Saldo is zichtbaar op elke pagina in de header.
- Saldo wordt realtime bijgewerkt na winst/verlies.
- Saldo blijft behouden na herladen of opnieuw inloggen.
- Bij 0 saldo verschijnt een optie voor daily bonus of refill (indien geactiveerd).

4. Als gebruiker wil ik toegang hebben tot alle beschikbare gokspellen, zodat ik kan kiezen wat ik wil spelen.
Prioriteit: Must
Acceptatiecriteria:
- Er is een overzichtspagina met alle spellen.
- Elk spel heeft een korte beschrijving en een “Speel nu”-knop.
- Spellen zijn alleen toegankelijk voor ingelogde gebruikers.
- Niet-beschikbare spellen tonen een duidelijke status (bijv. “Binnenkort”).

5. Als gebruiker wil ik mijn spelgeschiedenis kunnen bekijken, zodat ik inzicht heb in mijn resultaten.
Prioriteit: Nice
Acceptatiecriteria:
- Geschiedenis toont datum, speltype, inzet, resultaat en winst/verlies.
- Geschiedenis is filterbaar op datum en spel.
- Alleen de eigen geschiedenis is zichtbaar.
- Data wordt veilig opgeslagen en correct geladen.

6. Als gebruiker wil ik een daily bonus kunnen claimen, zodat ik kan blijven spelen als mijn saldo op is.
Prioriteit: Nice
Acceptatiecriteria:
- Daily bonus is één keer per 24 uur claimbaar.
- Na claimen wordt het saldo direct verhoogd.
- Timer toont wanneer de volgende bonus beschikbaar is.
- Bonusbedrag is configureerbaar in de backend
